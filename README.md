# Bank Customer Profitability Prediction

## Overview
This project is a Machine Learning-based Bank Customer Profitability Prediction system developed using Python and Jupyter Notebook. The project focuses on analyzing customer data, identifying profitable customers, and generating predictive insights using machine learning algorithms and data analysis techniques.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Customer Profitability Analysis
- Data Visualization using Charts and Graphs
- Machine Learning Model Training
- Prediction and Performance Evaluation

## Machine Learning Algorithms Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Import Dataset
2. Data Cleaning and Transformation
3. Exploratory Data Analysis
4. Feature Selection and Engineering
5. Train-Test Split
6. Model Training and Evaluation
7. Prediction and Result Analysis

## Objective
The main objective of this project is to predict customer profitability and help in understanding customer behavior using machine learning and data analytics techniques.
